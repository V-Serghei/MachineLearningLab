# AdaBoost Classification Project

## Setup Instructions

To ensure the correct functioning of the AdaBoost algorithm, you need to first extract the dataset from the archive.

### Step 1: Extract the dataset
The dataset is stored in a compressed archive file `creditcard.tar.xz`. Before running the code, extract it using the following command:

```bash
 tar -xvf creditcard.tar.xz
```

### Step 2: Install Dependencies
Ensure that all required Python dependencies are installed by running:

```bash
pip install -r requirements.txt
```

### Step 3: Run the Algorithm
After extracting the dataset, you can execute the AdaBoost classifier script:

```bash
python adaboost_classifier.py
```

## Notes
- Ensure that the extracted dataset is located in the correct directory as expected by the script.
- If the dataset is missing, download it again and repeat the extraction process.

Enjoy experimenting with AdaBoost! 🚀
